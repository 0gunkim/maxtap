# Bookmark Tab
- Save Bookmarks
  
    https://maxtap.vercel.app/

- vuejs composition api, typescript, tailwindcss, vite
  
  
## Setup
```
git clone https://github.com/0gunkim/maxtap.git
```

```
npm install
```
```
npm run dev
```
